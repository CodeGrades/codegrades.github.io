How it works

Code-grades are cumulative steps for learning how to code. They're a programming version of music grades, belts in martial arts or lifeguard certification. Level up by applying the knowledge and skills needed for each grade to your own fun, interesting and challenging coding projects.

Learn

Learn and prepare with the online curriculum, syllabus and resources for the code grade you're going for.

Grade

Take the code grade with a supportive “code mentor” who checks you have the skills and knowledge to level up.

Grow

Your code mentor (an experienced professional programmer) will give helpful feedback, support and advice.

Who is it for?

Learners

Code grades are a progressive path from your first steps in programming to an effective level of skill and knowledge needed for writing your own coding projects.

  • Code grades are created by experienced professional software developers: your skills and knowledge will be relevant, valuable and up-to-date.
  • When you go for grading you'll work with supportive experienced professional developers.
  • Like playing music, coding takes practice. Code grades give you the structure and goals needed for effective practice.
  • Code grades are fun for people of all ages and backgrounds.

Teachers

Wondering what to teach in coding lessons? Not sure what's relevant? Is the work of programmers a complete mystery to you? No idea what to do in "computer club"?

  • Code grades provide a teacher-friendly curriculum and syllabus.
  • They're student focused and encourage self-paced learning.
  • While code grades have been created by professional programmers, they also reflect feedback from teachers like you.
  • Grading is an opportunity for your students to shine, receive recognition and have their achievements celebrated.
  • Code grades help you gain the skills and knowledge needed to be an effective coding teacher.

Mentors

If you're a programmer you may think code grades are of no interest. But if you're a good programmer you already know that mentoring is a core part of developing your craft.

  • Mentoring beginners with easy-to-understand language and examples improves your own clarity of thought.
  • Your depth of understanding increases as you practice explaining, analogizing and summarizing difficult concepts.
  • Get paid to improve your pedagogical technique so you become an engaging mentor (and avoid being yet another patronising colleague).
  • Mentoring beginners is a sign of moral and professional growth.
